# Notebooks and IDEs (NAI01)

## Tools Used

These tools are commonly included as part of the standard toolkit for many professionals working with Python projects.

This course focuses on practical skills to install, configure, and navigate these tools effectively, rather than exhaustive feature coverage.

### **Anaconda**

- A comprehensive Python distribution service for data and solution developers, it manages Python packages and environments, preventing compatibility issues that can disrupt workflow.

### **VS Code**

- A versatile code editor, it provides a robust development environment that scales from simple scripts to complex applications, with excellent extension support for data work.

### **Jupyter Notebooks**

- An interactive computing environment for data science and machine learning, it enables interactive data manipulation by combining executable code, visualisations, and documentation in one interface.

- Ideal for exploratory data work, data flows, machine learning models, algorithms, and sharing results with stakeholders.
