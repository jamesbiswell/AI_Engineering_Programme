## Sentiment Analysis AI Project Objective
- Create a sentiment analysis model for movie reviews
- Use a pre-trained Hugging Face model to classify IMDb reviews as positive or negative
- Practice model loading, basic inference, and evaluation metrics without training a model
- Build a working classifier that can be tested with custom text

## Sentiment Analysis AI Project Steps
1. Set up the environment
2. Load a pre-trained model
3. Work with real data
4. Add interactive testing
5. Understand what you have built
6. Add proper error handling and edge cases
7. Create a visualisation of the results
8. Write a README documenting what you’ve learned
9. Push the project to GitHub as a portfolio example
